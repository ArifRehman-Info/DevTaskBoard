What is the difference between innerHTML, innerText, and textContent?

  • InnerHTML: innerHTML lets you see the HTML code inside a string, including things like spaces, line breaks, and
  • InnerText: This shows the "rendered" text of a node and understands styling and CSS. It’s helpful when you just want to see the content of the element without worrying about the formatting.
  • TextContent: It gets or sets the tag’s content as plain text, and it’s useful when you want to see the text along with the styling.

What is event delegation in the DOM?

Event delegation is when you attach one event listener to a parent element instead of adding one to each child. The parent handles events for all its kids, making things faster and working even for new elements added later.

What is event bubbling in the DOM?

Event bubbling is when an event starts on an element and then "bubbles up" to its parents, triggering their event listeners. It’s useful for handling events at the parent level.

How do you create, add, and remove elements using JavaScript?

  • Create an element: document.createElement('tagName');
  • Add an element: parentElement.appendChild(newElement);
  • Remove an element: element.remove();
